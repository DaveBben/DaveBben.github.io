Amazon Echo Show Web Browser

Posted on July 20, 2017  |   1 minutes  | dave

The Amazon Echo Show has a hidden web browser, and here is how you can enable it. Keep in mind that the Amazon Echo Show is running Android at its core. Therefore, like any normal Android web browser, you have access to the same WebKit features and vulnerabilities when using the browser.

Enable Amazon Echo Show Web Browser

Go into your settings and go to Legal and Compliance.

Then go into Amazon Privacy Notice.

From there, scroll to the section that says Information We Collect. You should find a link that says My Account. Clicking on it will take you to Amazon’s homepage. From there, just search for any outbound link. I searched for the Google app, then went to their privacy policy page which eventually led me to Google’s home page.
